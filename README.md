### 📌 Featured Projects

**🎓 AI-Based Academic Approval System for UGC & AICTE** — *SIH Grand Finals 2025*
Full-stack workflow automation platform (React, Node.js, Express, MongoDB) that eliminates manual bottlenecks in institutional approval management. Integrated **Llama 3 + LangChain** for automated data extraction from PDF/JPG documents, and polynomial regression to forecast 5-year enrollment trends via Recharts dashboards. Role-based access across institutions, regulators, and reviewers — MVP shipped in a 36-hour sprint, earning Grand Finalist standing among 826,635+ participants.

**⚡ DPI Engine — Multi-Threaded Deep Packet Inspection System** — *C++17* · [Source Code](https://github.com/AchieverSana/Deep_Packet_Inspection)
Multi-threaded packet-processing pipeline (Reader → Load Balancer → Fast Path) with configurable thread pools and bounded thread-safe queues for zero packet loss. Implements TLS SNI + HTTP Host-header parsing to classify 17+ applications (YouTube, Netflix, Discord, TikTok) without decrypting traffic, plus stateful flow tracking via a five-tuple hash map. Validated against a 462,000-packet synthetic PCAP capture with zero processing errors.

**✍️ Let's Blog — Full-Stack Blogging Platform** — [Source Code](https://github.com/AchieverSana/Let-sBlog)
Admin-only post authoring with asymmetric comment permissions enforced at the middleware level. Hardened with httpOnly-cookie JWT sessions, bcrypt hashing, input sanitization, and rate limiting to mitigate XSS/injection/brute-force attacks. Eliminated N+1 queries via batched lookups and MongoDB indexes. CI/CD pipeline with 30 automated tests (Vitest, React Testing Library) + ESLint on every push.

**🔗 URL Shortener** — [Source Code](https://github.com/AchieverSana/Url_Shortener)
URL shortener built with Node.js, Express & MongoDB — JWT auth, role-based access (Admin/Normal), and click tracking.
